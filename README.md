# Next.js Portfolio Starter (App Router)

A config-driven portfolio template migrated from a single-page HTML project into a reusable Next.js App Router structure.

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Customize

1. Update profile and section data in `src/data/config.ts`.
2. Update site and navigation settings in `config/settings.yaml`.
3. Replace `public/avatar-placeholder.svg` and branding assets under `public/`.

## Quality Commands

```bash
npm run lint
npm run test
npm run test:integration
npm run test:e2e
npm run build
```

## API + CLI Entrypoints

- API endpoint: `GET /api/portfolio`
- CLI summary: `npm run summary`
